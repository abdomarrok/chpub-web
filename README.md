# Ch.Pub — Agence Publicitaire

Une Single-Page Application (SPA) haute performance pour une agence de publicité, incluant un configurateur de panneaux Alucobond en 3D CSS et un système de génération de devis en direct.

## 🚀 Fonctionnalités

- **Performances Flash** : Développement en Vanilla JS/CSS pour un chargement instantané.
- **Configurateur Alucobond 3D CSS** : Visualisation en temps réel du panneau publicitaire avec calcul dynamique du prix.
- **Système de Devis** : Génération de devis PDF/Printable en direct avec export WhatsApp.
- **Design Premium** : Thème sombre, accents bordeaux et or, animations fluides et typographie soignée.
- **100% Responsive** : Optimisé pour tous les écrans (Mobile/Tablette/Desktop).
- **Architecture Propre** : Séparation stricte des préoccupations (Logic/Design/Config).

## 🛠️ Stack Technique

- **HTML5** : Structure sémantique et SEO friendly.
- **CSS3** : Variables CSS, Grid, Flexbox, et 3D Transforms (pas de librairies externes).
- **JavaScript (ES6+)** : Modules, Fetch, et logique métier indépendante.
- **Config Driven** : Toute l'agence est paramétrable via `js/config.js`.
